# Assembly Language Database Project

This project is developed for CS-207 Computer Organization & Assembly Language.

## Description
A lightweight database system implemented using 8086 assembly language to manage population and resource data for remote areas with limited internet connectivity.

## Features
- Add records
- Prevent duplicate Sr#
- Sort records
- Calculate totals

## Tools
- EMU-8086
- 8086 Assembly Language

## University
HITEC University, Taxila
